# SlotSwapper

A peer-to-peer time-slot scheduling application built for a web development assignment.

## Tech Stack
* **Frontend:** React
* **Backend:** Node.js / Express
* **Database:** PostgreSQL
* **Auth:** JWT (JSON Web Tokens)

## How to Run
1.  `cd backend` and run `npm install`.
2.  Create a `.env` file and set the `DATABASE_URL`.
3.  `cd frontend` and run `npm install`.
4.  Run `npm start` in the backend and `npm run dev` in the frontend.
